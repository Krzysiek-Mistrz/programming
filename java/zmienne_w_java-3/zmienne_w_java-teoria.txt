4) Zad "Paszport samochodowy"
Wyswietl wlasnicwosci pojazdu. Napisz program wypisujacy wlasciwosci samochodu.
Zadanie:
Naprawic kod tak zeby wyswietlal wlasciwosci dango samochodu.
Strumien wyjscia:
name: Toyota
Engine: 4.7
Rok: 2019
Zwroc uwage na typy zmiennych, powinny byc one dobrze i prawidlowo dobrane.
1) Rozwiazanie:
public class Program
{
    public static void main(String[] args)
    {
        String name = "Toyota";
        double engine = 4.7;
        int year = 2019;

        System.out.println("Name: " + name);
        System.out.println("Engine: " + engine);
        System.out.println("Year: " + year);
    }
}

21
false

{
