1) Enum to specjalny typ uzywany do definiowania kolekcji stalych. Oto przyklad:
public enum Rank {
  SOLDIER,
  SERGANT,
  CAPTAIN
}
Zauwaz ze wartosci oddzielone sa przecinkami. Mozesz odwolywac sie do stalych w powyzszym wyliczeniu za pomoca skladni z kropka tak tak np. wywolywales dana metode na rzecz danego obiektu :) Przyklad:
Rank a = Rank.SOLDIER;
Zasadniczo typ Enum sluzy do definiowania zmiennych reprezentujacych okreslony zestaw danych.

3) Po zadeklarowaniu Enum mozemy sprawdzic odpowiednie wartosci za pomoca instrukcji switch na przyklad.
Rank a = Rank.SOLDIER;

switch(a) {
  case SOLDIER:
    System.out.println("Soldier says hi!");
    break;
  case SERGEANT:
    System.out.println("Sergeant says hello!");
    break;
  case CAPTAIN:
    System.out.println("Captain says welcome!");
    break;
}
Wyswietli sie w rezultacie wykonania powyzszego kodu komunikat: Soldier says hi!

8) Zadanie (praktyka)
Podaj rezultat ponizszego kodu:
enum Color {
  RED, BLUE, GREEN
}
class PrintColor {
  public static void main(String[] args) {
    Color color = Color.RED;
    switch(color) {
      case BLUE:
        System.out.println("1");
      break;
      case GREEN:
        System.out.println("2");
      break;
      case default:
        System.out.println("0");
      break;
    }
  }
}
Odpowiedz: Wykona sie przypadek default poniewaz nie ma uwzglednionej opcji ze argument bedacy zmienna typu enum = Color przjmie wartosc = "RED".
