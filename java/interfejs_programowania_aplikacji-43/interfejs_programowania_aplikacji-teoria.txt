1) API dla Java to zbior klas, interfejsow, ktore zostaly napisane do uzytkowania w API. Dokumentacje Java API mozna znalezc na stronie Oracle pod adresem: http://docs.oracle.com/javase/8/api/ Po zlokalizowaniu pakietu ktorego chcesz uzyc, musisz go zaimportowac do swojego kodu. Pakiet mozna zaimportowac za pomoca slowa kluczowego import, ktore juz przeciez poznawalismy, pamietasz moze import java.util.Scanner;? Przyklad:
import java.awt.*;
Pakiety awt zawiera wszystkie klasy do tworzenia interfejsow uzytkownika i do malowania grafik i obrazow. Symbol wieloznaczny (*) sluzy natomiast do importowania wszystkich klas w pakiecie.

Uzupelnij luke w kodzie tak by zaimportowac wszystkie klasy w pakiecie awt.
