1) Java obsluguje zagniezdzanie klas. Klasa moze byc czlonkiem innej klasy. Tworzenie klasy wewnetrznej jest stosunkowo proste. Po prostu nalezy napisac klase wewnatrz innej klasy ;) W przeciwienstwie do zwyklej klasy klasa wewnetrzna moze byc prywatna. Po zadeklarowaniu wewnetrznej klasy prywatnej nie mozna uzyskac do niej dostepu z obiektu spoza klasy.
Przyklad:
class Robot {
  int id;
  Robot(int i) {
    id = i;
    Brain b = new Brain();
    b.think();
  }
  private class Brain {
    public void think() {
      System.out.println(id + " is thinking");
    }
  }
}
Klasa Robot ma zatem teraz klase wewnetrzna o nazwie Brain. Klasa wewnetrzna moze uzyskac dostep do wszystkich zmiennych skladowych i metod swojej klasy zewnetrznej, ale nie mozna uzyskac do niej dostepu z zadnej klasy zewnetrznej.
