Zadanie "Agencja turystyczna"
Uzupelnij kod zeby otrzymac dystans w kilometrach i metrach w nowych liniach. Jestes menadzerem agencji turystycznej i musisz oblicyc dystans miedzy dwma miastmi. podany program bierze jako wartosc podana dystans miedzy dwoma miastami jako input.
Zadanie:
Uzupelnij kod zeby wyswietlic dystans w kilometrach i metrach w nastepujacych liniach.
Sample input:
3644
Sample Output:
3
644
Wyjasnienie:
1 km jest to 1000 m, zatem 3644m bedzie stanowic 3km i 644m.
Skorzystaj z operatora / w celu obliczenia ilosci kilometrow i operatora % w celu obliczenia metrow.
1) Rozwiazanie:
import java.util.Scanner;
public class Program
{
    public static void main(String[] args) 
    {
        Scanner Object = new Scanner(System.in);

        int distance = Object.nextInt();

        int kilometers = distance / 1000;
        int meters = distance % 1000;

        System.out.println(kilometers);
        System.out.println(meters);
    }
}

Zad "Czas na zadanie"
Napisz program ktory pobierze czas spedzony nad azdym przedmiotem i obliczy calkowity czas potrzebny ogolnie na wykonanie wszystkich zadan. Studenci otrzymali zadanie z historii matmy i geometrii. Napisz program ktory pobierze jak input czas spedzony przez studentow nad kazdym zadanie z kazdego przedmiotu i policzy calkowity czas w godiznach i minutach jaki studenci spedzili nad danym przedmiotem. Wynik powinien zostac wyswietlony w kazdej nastepnej linii.
Sample Input
35
40
39
sample output
1
54
Wyjasnienie: Calkowity spedzony czas w minutach jest = 114, ktory jst rowny 1h i 54 minuty. Uzyj typu calkowitego int do zmiennych reprezentujacych godziny i minuty.
1) Rozwiazanie:
import java.util.Scanner;
public class Program
{
    public static void main(String[] args)
    {
        Scanner scanner = new Scanner(System.in);

        int math = scanner.nextInt();
        int history = scanner.nextInt();
        int geometry = scanner.nextInt();

        int sum = math + history + geometry;
        int hours = sum / 60;
        int minutes = sum % 60;

        System.out.println(hours);
        System.out.println(minutes);
    }
}

zad "2PIr"
Napisz maly program ktory policzy i wyswietli obwod kola dla ciebie. Podczas twojej lekcji matematyki zostales poproszony o wyliczenie obwodu kola. postanowiles napsac krotki program ktory pobierze promien okregu jako input, nastepnie policzy obwod tego kola za ciebie. Wartosc pi zostala zdefiniowana odgornie.
Sample input:
4
Sample output:
25.12
Wskazowka: obwod kola dany jest wzorem 2*pi*r.
1) Rozwiazanie:
import java.util.Scanner;
public class Main
{
    public static void main(String[] args)
    {
        Scanner Rad = new Scanner(System.in);
        
        double radius = Rad.nextDouble();
        double pi = 3.14;
        double perimeter = 2 * pi * radius;
        
        System.out.println(perimeter);
    }
}
