5) Dobrze przejdzmy teraz do instalacji srodowiska programistycznego IDE. Zacznamy od oficjalnej strony Oracle i sciagamy z niej JDK - Java Development Kit, sluzacy za instalcji podstawowych bibliotek jezyka Java do naszego systemu. Podaje jeszcze najnowszego linka do JDK SE 17.01: "https://www.oracle.com/java/technologies/javase/jdk17-archive-downloads.html", jednak jesli czytasz to pare lat pozniej, wpisz w wyszukiwarke kolejno JDK SE 18, JDK SE 19, JDK SE 20 i tak dojdziesz do najnowszej wersji. Naszym kompilatorem bedzie srodowisko o nazwie Intelli Idea: "https://www.jetbrains.com/idea/", od firmy JetBrains. Tak tej samej ktora wydala PyCharm-a.
Po instalcji dodajemy Zmienna srodowiskowa naszego JDK do zmiennych srodowiskowych, w tym celu wpisujemy w lupce w windowie zmienne srodwiskowe, otieramy klikamy na path i dodajemy nowa sciezke prowadzaca do folderu instalacji naszego JDK dokladni do jego folderu bin. Od teraz mozemy programu w javie uruchamiac z poziomu konsoli.
