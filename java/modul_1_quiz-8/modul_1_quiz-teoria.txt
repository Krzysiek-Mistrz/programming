4) Zadanie "Konwerter czasu"
Potrzebujesz wykonac program zamieniajacy dni na sekundy. Twoj kod powinien pobrac o uzytkownika liczbe dni i zamienic ta liczbe dni na odpowiadajaca mu liczbe sekund.
Sample input:
12
Sample output:
1036800
Rozwiazanie:
import java.util.Scanner;
public class Program
{
  public static void main(String[] args)
  {
    Scanner scanner = new Scanner(System.in);
    int days = scanner.nextInt();
    
    int hours = days * 24;
    int minutes = hours * 60;
    int seconds = minutes * 60;

    System.out.println("Liczba sekund: " + seconds);
  }
}
