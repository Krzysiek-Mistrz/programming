5) Pora juz zeby omowic kolejne niezwykle wazne slowo kluczowe public zaraz na poczatku kazdej klasy czy funkcji main() - public i takze inne ciekawe modyfikatory dostepu.
public static void main(String[] args) {
    ...
}
Public to modyfikator dostepu co oznacza ze sluzy do ustawiania poziomu dostepu. Mozesz uzywac modyfikatorow dostepu do klas, atrybutow i metod.
W przypadku klas dostepne modyfikatory dostepu sa - publiczne lub domyslne (pozostawione puste), zgodnie z opisem ponizej:
public: klasa jest dostepna dla kazdej klasy.
default: klasa jest dostepna tylko dla klas w tym samym pakiecie.
W przypadku atrybutow i metod dostepone sa nastepujace opcje:
default: zmienna lub metoda zadeklarowana jest bez modyfikatora kontroli dostepu jest dostepna dla dowolnej innej klasy w tym samym pakiecie.
public: dostepne z dowolnej innej klasy.
protected: zapewnia ten sam dostep co domyslny modyfikator dostepu, z dodatkiem ze podklasy maja dostep do chronionych metod i zmiennych nadklasy (W nastepnych lekcjach).
private: dostene tylko w obrebie samej zadeklarowanej klasy.
Przyklad:
public class Vehicle {
  private int maxSpeed;
  private int wheels;
  privte String color;
  private double fuelCapacity;

  public void horn() {
    System.out.println("Beep");
  }
}
Najlepsza praktyka jest zeby zmienne wewnatrz klasy byly prywatne. Zmienne sa dostepne i modyfikowane za pomoca  Getter-ow i Setter-ow. Zaraz zreszta o nich odrobine wiecej powiemy.
