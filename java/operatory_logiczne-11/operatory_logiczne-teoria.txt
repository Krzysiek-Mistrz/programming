4) Operator OR. Operator OR, czyli || sprawdza czy ktorykolwiek z warunkow jest prawdziwy. Caly warunek wowczas staje sie prawdziwy, jezeli tylko ktorykolwiek z operandow staje sie prawdziwy.
Dla przykladu:
int age = 25;
int money = 100;
if(age > 18 || money > 500) {
  System.out.println("Welcome");
}
Kod powyzszy wyswietli "Welcome" jezeli wiek jest wiekszy od 18 lub jezelipieniadze sa wieksze od 500.
Operator NOT czyli ! jest operator logiczny sluzacym do odwrocenia stanu logicznego swoich operandow, jezeli warunek jest prawdziwy, operator NOT zrobi z niego falszywy.
int age = 25;
if(!(age > 18))
{
  System.out.println("To young");
}
else
{
  System.out.println("Welcome");
}
Czytamy warunek nasteoujaco: Jezeli age jest NIE wiekszy niz 18.
