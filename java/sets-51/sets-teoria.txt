1) Set to kolekcja ktora nie moze zawierac zduplikowanych elementow. Modeluje ona abstrakcje zbiorow matematycznych. Jedna z implementacji Set jest klasa HashSet. Przyklad:
import java.util.HashSet;
public class MyClass {
  public static void main(String[] args) {
    HashSet<String> set = new HashSet<String>();
    set.add("A");
    set.add("B");
    set.add("C");
    System.out.println(set);
  }
}
Mozesz uzyc metody size() w celu otrzymania liczby elementow kontenera klasy HashSet.

2) Zadanie (praktyka)
import java.util.HashSet;
class A {
  public static void main(String[] args) {
    HashSet<String> set = new HashSet<String>();
    set.add("A");
    set.add("B");
    set.add("C");
    System.out.println(set.size());
  }
}
Odpowiedz: 3
Wyjasnienie: Metoda size() zwraca liczbe elementow w danym kontenerze.

Teraz o tym co to jest hashowanie? Tablica mieszajaca (tablica mieszajaca) przechowuje informacje za pomoca mechanmizmu zwanego mieszaniem, w ktorym zawartosc informacyjna klucza jest uzywana do okreslenia unikalnej wartosci zwanej kodem mieszajacym. Zasadniczo kazdy element w HashSet jest powiazany z unikalnym kodem skrotu.
