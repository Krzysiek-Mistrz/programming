5) Zadanie Sprawdzajace Z Modulu 2:
Bierzesz pozyczke od znajomego i musisz obliczyc ile bedziesz mu winien po 3 miesiacach, co miesiac bedziesz splacac mu 10% pozostalej kwoty kredytu. Utworz program ktory pobiera kwote pozyczki jako dane wejsciowe, oblicza i wyswietla pozostala po 3 miesiacach kwote do oddania.
Przykladowe dane wejciowe:
20000
Przykladowe dane wyjsciowe:
10628
Oto miesieczny harmonogram z platnosci:
Miesiac 1 - platnosc 10% z 20000 = 2000, pozostalo - 18000
Miesiac 2 - platnosc 10% z 18000 = 1800, pozostalo - 16200
Miesiac 3 - platnosc 10% z 16200 = 1620, pozostalo - 14580
Uzyj petli do obliczenia podanej wartosci w kolejnych miesiacach, ponadto do przechowywania danych staraj sie uzywac typow int
1) Rozwiazanie:
import java.util.Scanner;
public class Program
{
  public static void main(String[] args) {
    Scanner scanner = new Scanner(System.in);
    int amount = scanner.nextInt();
    int remaining = amount;
    for (int i = 0; i < 3; i ++)
    {
      int payment = (amount * 10)/100;
      remaining = amount - payment;
      amount = remaining;
    }
    System.out.println(remaining)
  }
}
